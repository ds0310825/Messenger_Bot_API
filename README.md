# Messenger_Bot_API (for Group Chat)

非官方API  
用於群組聊天室  
輸入關鍵字會自動回覆  
並增加了"搜尋圖片"等內建功能  

為舊作品的延伸  
增加了UI介面 及 更容易的操作  

使用時需有FaceBook帳號 和 確認是否已加入該聊天室  

EXE檔可以於下面的網址下載:
